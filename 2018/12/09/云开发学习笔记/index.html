小程序开发入门

用云函数新增一条数据，

在云函数中初始化云数据库

cloud.init()

const db = cloud.database({
env:"环境id"
})
const productsCollection = db.collection("products")

然后用：productsCollection.add({data:{}})。新增数据。

onLoad: function (options) {
productsCollection.get().then(res => {

下拉刷新功能：



json中加入：

“enablePullDownRefresh”:true,

js中写入onPullDownRefresh函数：

onPullDownRefresh: function () {
productsCollection.get().then(res => {
this.setData({
products: res.data
},res=>{
console.log(“cg”)
wx.stopPullDownRefresh()
})
})
},

（其中，wx.stopPullDownRefresh()简单理解是缩短刷新数据）

上滑载入新数据

（微信查询一次数据最多返回20条）：

json中：

“onReachBottomDistance”:100

其中，100代表一共载入到第100条

js中：


onReachBottom: function () {
let page =this.data.page+20

productsCollection.skip(page).get().then(res=>{
let old_data=this.data.products
let new_data=res.data

this.setData({
products:old_data.concat(new_data),
page:page
},res=>{
console.log(res)
})
})

},

(let 用于在函数中定义变量，变量作用域为本函数。concat用于拼接数据)



条件查询


挑选出color为red 的数据

productsCollection.where({
color:”red”
}).get().then(res => {
this.setData({
products: res.data
})
})

挑选出price在40到50之间的

productsCollection.where({
price:.lt(50).and(.gt(40))
}).get().then(res => {
this.setData({
products: res.data
})
})

查询两条数据

productsCollection.limit(2).get().then(res => {
this.setData({
products: res.data
})
})

价格降序（asc：升序 ）

productsCollection
.orderBy(“price”,”desc”)
.get().then(res => {
this.setData({
products: res.data
})
})

统计数据量

productsCollection.count().then(res => {
this.setData({
count: res.total

})
})
